# Advanced HTML

Create and structure web pages using HTML5 semantic tags.

## Description

This project covers the basics of HTML: doctype, page skeleton, semantic tags,
headings, lists, tables, media, and embedded content.

Company name used across pages: **Techium**.

| `3-index.html` | Simple header, main, and footer sections |
