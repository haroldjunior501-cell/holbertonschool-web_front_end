# CSS_advanced

## Techium — Homepage

Static homepage built with HTML5 and advanced CSS3, as part of the Holberton School / Holberton x Holberton web front-end curriculum.

### Project goal

Recreate the Techium homepage design using semantic HTML and a hand-written CSS stylesheet (no framework), covering:

- Selectors, properties and values
- Block vs inline styling
- CSS reset for cross-browser consistency
- CSS custom properties (variables)
- Inline, embedded and external CSS
- A float-based grid system (`row`, `col-1-2`, `col-1-3`)
- Pseudo-classes vs pseudo-elements
- Background gradients
- CSS animations and 2D/3D transforms
- Vendor prefixes
